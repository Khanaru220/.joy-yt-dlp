# My joyful 🏡 home to 🤖 automate daily chores

## Import and run script in different folder

with `.js`

```js
const functionName = require('./path/to/that/JSfile');
functionName();
```

with `.bat`

```bat
call ".\path\to\that\batFile.bat"
pause
```
